# Project Notes - Node-X

Here are some details on the project architecture:

- **Express.js Server**: Listens on port 3000 and exposes REST APIs.
- **Vanilla CSS styling**: High performance, custom theme, neon red-blue accents.
- **Archiver Integration**: Dynamically bundles directory trees into standard ZIP archives on request.

## Color Palette Configuration
- Background: `#030712` (Vibrant Dark slate/gray)
- Neon Blue: `#00b4d8`
- Neon Red: `#ff0054`
- Glassmorphism overlays: `rgba(255, 255, 255, 0.03)` with `backdrop-filter: blur(12px)`
